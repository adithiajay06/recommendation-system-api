# 🚀 Complete Recommendation System API

A production-ready recommendation system built with FastAPI, SQLite, and SQLAlchemy.  
This project demonstrates how modern recommendation engines work, including user behavior tracking, personalized recommendations, cold-start handling, and performance evaluation.

---

## 📌 Overview

This system provides personalized content recommendations to users based on their interactions, preferences, and similarity patterns.

It simulates real-world platforms like Netflix, YouTube, and Amazon by:
- Tracking user interactions
- Learning preferences
- Suggesting relevant content

---

## 🧠 Features

- ✅ REST API built with FastAPI
- ✅ SQLite database with normalized schema
- ✅ Repository pattern (clean architecture)
- ✅ User, Content, Skills, Interaction tracking
- ✅ Cold-start recommendation handling
- ✅ Caching for fast responses
- ✅ Recommendation explanations
- ✅ Evaluation metrics:
  - Precision@5
  - Recall@5
  - NDCG@5
- ✅ Unit testing with pytest
- ✅ Request logging & performance monitoring
